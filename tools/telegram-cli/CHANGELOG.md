# Changelog – telegram-cli

All notable changes to this tool are documented in this file.

## [1.0.0] – 2025-11-27

- Initial release:
  - Duration-based reminders (30s, 5m, 1h, 1h30m)
  - Time-based reminders (ISO format)
  - Background execution via nohup
  - Auto-load from ~/AGENTS.env
  - macOS compatible (avoids `at` issues)
