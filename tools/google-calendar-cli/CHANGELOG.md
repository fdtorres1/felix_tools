# Changelog – google-calendar-cli

All notable changes to this tool are documented in this file.

## [1.0.0] – 2025-11-27

- Initial release with full feature set:
  - OAuth authentication flow
  - List calendars
  - List events with date range filters
  - Create timed and all-day events
  - Update events by ID
  - Delete events
  - Quick add (natural language)
  - Google Meet link generation
  - Attendees with send-updates control
  - Attendees from Contacts labels
  - Recurring events (RRULE builder)
  - Bulk create from JSON/JSONL
  - Free/busy lookup
  - Dry-run mode
  - Timezone support
  - Telegram notifications on failure
