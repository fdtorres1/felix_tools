# Changelog – eventkit-calendar

All notable changes to this tool are documented in this file.

## [1.0.0] – 2025-11-27

- Initial release with full feature set:
  - List calendars with IDs and sources
  - List events with date range filters
  - Find events by title (exact or contains)
  - Create timed and all-day events
  - Update events by ID
  - Delete events by ID
  - Bulk create from JSONL files
  - Daily digest with Telegram integration
  - Weekly digest (7-day view)
  - Calendar filtering by ID or name
  - JSON output for programmatic use
  - Auto-load configuration from ~/AGENTS.env
