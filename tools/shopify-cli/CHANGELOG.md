# Changelog – shopify-cli

All notable changes to this tool are documented in this file.

## [1.0.0] – 2025-11-27

- Initial release with full feature set:
  - Auth verification
  - Products search and export
  - Customers search and export
  - Orders search and export
  - Fulfillment orders listing
  - Fulfillment deadline setting
  - Metafield get and set
  - Pages listing
  - Blogs management
  - Articles management
  - Collections management
  - JSONL and CSV exports
  - Cursor-based pagination
  - Dry-run mode for mutations
  - Custom GraphQL query support
