# Changelog – clickup-cli

All notable changes to this tool are documented in this file.

## [1.0.0] – 2025-11-27

- Initial release with full feature set:
  - Teams, spaces, lists, and statuses browsing
  - Task search with filters (status, assignee, tag, dates)
  - Task CRUD operations (create, update, close)
  - Bulk create from JSON/JSONL files
  - Import with name-based resolution
  - Export to JSONL and CSV
  - Comments and tags management
  - Custom fields viewing
  - Cleanup/batch close operations
  - Name-based resolution for teams, spaces, lists, users
  - Status and priority resolution by name
  - Dry-run mode for all mutations
  - Optional Telegram and macOS notifications
