# Changelog – handwrytten-cli

All notable changes to this tool are documented in this file.

## [Unreleased]

## [0.1.0] – 2025-11-27

- Initial version of `handwrytten-cli` with:
  - Card listing with optional category filtering and image inclusion
  - Font (handwriting style) listing
  - Category listing
  - Card detail retrieval
  - Single-step order creation for sending cards
  - Order detail retrieval
  - Order cancellation
  - Address book listing and entry creation
  - JSON and table output formats
  - Environment-based configuration with API key support
  - Dry-run mode for testing without sending actual orders
