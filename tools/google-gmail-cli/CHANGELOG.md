# Changelog – google-gmail-cli

All notable changes to this tool are documented in this file.

## [1.0.0] – 2025-11-27

- Initial release with full feature set:
  - OAuth authentication flow
  - Profile lookup
  - Message listing and retrieval
  - Send text and HTML emails
  - Reply and reply-all
  - Draft creation, sending, deletion
  - Label management (list, create, apply, remove)
  - Alias (sendAs) management
  - Contacts label recipient expansion
  - Scheduled send queue with LaunchAgent support
  - Queue update and cancel
  - Dry-run mode for all mutations
  - Telegram notifications on queue failures
