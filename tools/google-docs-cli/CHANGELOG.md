# Changelog – google-docs-cli

All notable changes to this tool are documented in this file.

## [1.0.0] – 2025-11-27

- Initial release with full feature set:
  - OAuth authentication flow
  - Read document as text
  - List headings (levels 1-6)
  - Find and replace (case-sensitive option)
  - Append paragraphs
  - Insert headings with styling
  - Insert tables
  - Insert images (inline)
  - Insert page breaks
  - Insert section breaks
  - Target insertion after specific headings
  - Create new documents
  - Export to PDF/text via Drive
  - Share documents
  - Retry with exponential backoff
  - Telegram notifications on failure
