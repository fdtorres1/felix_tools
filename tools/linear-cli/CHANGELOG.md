# Changelog – linear-cli

All notable changes to this tool are documented in this file.

## [1.0.0] – 2025-11-27

- Initial release with full feature set:
  - Raw GraphQL execution
  - Team listing and ID lookup
  - Label listing (with team filter)
  - User ID resolution by email/name
  - Issue search by title
  - Issue creation with labels
  - Issue update (title, description, labels)
  - Comment creation
  - Dry-run by default for mutations
  - Auto-load from ~/AGENTS.env
