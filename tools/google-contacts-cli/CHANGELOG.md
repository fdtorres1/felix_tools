# Changelog – google-contacts-cli

All notable changes to this tool are documented in this file.

## [1.0.0] – 2025-11-27

- Initial release with full feature set:
  - OAuth authentication flow
  - List contacts with field masks
  - Search by query (name/email/phone)
  - Get contact by resource name
  - Create contacts with details
  - Update contacts (auto etag handling)
  - Delete contacts
  - List contact groups (labels)
  - Create contact groups
  - Add/remove contacts to/from groups
  - Get all emails from a group
  - Delete groups
  - Telegram notifications on failure
